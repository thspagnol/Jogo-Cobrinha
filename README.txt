Jogo da Cobrinha em Python (Pygame)

Este é um simples e divertido Jogo da Cobrinha criado com Python e Pygame! O objetivo é guiar a cobra pela tela, comer maçãs para crescer e evitar colisões com obstáculos e as bordas da janela.

Características:
- Movimentação da cobra usando as setas do teclado
- Geração aleatória de maçãs e obstáculos
- Aumento de velocidade conforme a pontuação cresce
- Exibição de pontuação na tela
- Detecção de colisões e sistema de Game Over

Como executar:
- Certifique-se de ter o Python instalado.
- Instale as dependências:
pip install pygame
- Execute o jogo:
python jogo_da_cobrinha.py
